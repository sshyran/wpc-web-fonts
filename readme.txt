Easily load multiple Google web fonts and multiple custom web fonts to use in your theme files.

== Description ==

This plugin makes it easy for any web developer to append user selected Google web fonts and custom uploaded web fonts to your font dropdown list in your theme options panel.

[Live Demo & Documentation](http://webplantmedia.com/starter-themes/wordpresscanvas/features/plugins/wpc-web-fonts/)

== Installation ==

1. Uzip the `wpc-web-fonts.zip` folder.
2. Upload the `wpc-web-fonts` folder to your `/wp-content/plugins` directory.
3. In your WordPress dashboard, head over to the *Plugins* section.
4. Activate *WordPress Canvas Shortcodes*.
